# 달러 투자 적정 시기 판단 웹앱

현재 환율 데이터를 활용하여 달러 투자의 적정 시기를 판단하는 React 기반 웹앱입니다.

## 기능

- **투자 적정 시기 판단**: 4가지 조건을 체크하여 투자 적합성을 판단
- **실시간 환율 정보**: 현재 원/달러 환율 표시
- **달러 지수 모니터링**: 달러 지수 및 52주 평균 비교
- **달러 갭 비율 계산**: 환율과 달러 지수를 기반으로 한 갭 비율 분석
- **52주 평균 데이터**: 최근 52주간의 최저/최고/평균 데이터 제공

## 투자 적정 시기 판단 조건

다음 4가지 조건을 모두 만족할 때 "투자 시작 적합"으로 판단합니다:

1. 현재 원/달러 환율 < 52주 평균 환율
2. 현재 달러 지수 < 52주 평균 달러 지수
3. 현재 달러 갭 비율 > 52주 평균 달러 갭 비율
4. 현재 원/달러 환율 < 적정 환율

## 계산 공식

- **달러 갭 비율** = (원/달러 환율) ÷ (달러 지수) × 100
- **적정 환율** = (52주 평균 달러 갭 비율) × (현재 달러 지수) ÷ 100

## 기술 스택

```bash

npm install

npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 데이터 소스

- **환율 데이터**: GitHub에서 제공하는 환율 데이터 (`data/exchange-rates.json`)
- **달러 지수**: Alpha Vantage API 또는 FRED API (GitHub Actions로 자동 수집)
- **52주 평균**: GitHub Actions로 자동 계산

## 배포

GitHub Pages를 통해 자동 배포됩니다. `main` 브랜치에 푸시하면 자동으로 빌드 및 배포가 진행됩니다.

## 참고 사항
